## Running instructions

1. Clone the repo
```text
git clone --depth=1 
```

1. Start the App
```text
npm start
```

1. Run in Developer Mode

This requires `nodemon`
Install `nodemon` using `npm i nodemon -g`

```text
npm dev
```

## Code
- UI code is in `app` folder
- Backend code is `backend` folder
